# README

## SUPERHEROES API

Here goes a rails API comprising heroes, hero_powers(joint table) and powers. The characters used are from the comic and Amazon Prime's TV show entitled 
#### The Boys.

## ENDPOINTS

The valid endpoints for the API include:
* GET /heroes
* GET /heroes/:id

* GET /powers
* GET /powers/:id

* PATCH /powers/:id
* POST /hero_powers
